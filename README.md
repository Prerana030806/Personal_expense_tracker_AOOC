# Personal_expense_tracker
Personal Finance Tracker (Java) is a simple console app to track income, expenses, and generate monthly reports.
---

## 🎯 Objective
To develop a simple system that helps users track and manage their personal finances efficiently.

---

## 🔮 Future Enhancements
- GUI using Java Swing / JavaFX
- Database integration (MySQL)
- Graphs and analytics dashboard
- Mobile/Web version
